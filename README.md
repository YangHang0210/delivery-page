# 骑手配送模拟页面

手机端模拟骑手配送中的取件页面，纯前端实现，无需后端服务。

## 快速使用

### 方式一：直接打开（推荐）

用手机浏览器（Safari / Chrome）直接打开 `index.html` 文件。

### 方式二：本地服务器

```bash
cd delivery-page
python3 -m http.server 8080
```

手机与电脑在同一 WiFi 下，浏览器访问 `http://<电脑IP>:8080`。

### 方式三：添加到主屏幕

在 Safari 中打开页面 → 点击分享按钮 → "添加到主屏幕"，图标和标题会更像原生 App。

## 自定义设置

**长按页面顶部标题栏**（约 1 秒）即可打开设置面板，可配置：

| 配置项 | 说明 |
|--------|------|
| 送达地址（小区名称） | 你的小区名，如"阳光花园" |
| 送达详细地址 | 楼栋门牌号，如"2栋1单元502室" |
| 商家名称 | 留空则每次随机展示不同商家 |
| 商家地址 | 留空则随机 |
| 配送剩余分钟数 | 留空则 20~35 分钟随机 |

设置保存在浏览器 localStorage 中，下次打开自动生效。

## 页面特性

- 每次打开自动生成新的订单号、商品、顾客信息
- 实时倒计时，预计送达时间动态计算
- 动画地图路线展示
- 点击"确认送达"可模拟完成动画并自动刷新新订单
- 底部导航栏、通知角标等细节还原

## 技术栈

纯 HTML + CSS + JavaScript，零外部依赖，单文件即可运行。
